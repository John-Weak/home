## What is this?
Personal portfolio website I built to establish my online presence.

# Tech
- Nuxt.js
- TailWind CSS
- Typescript


## Setup

Make sure to install the dependencies

```bash
yarn install
```

## Development

Start the development server on http://localhost:3000

```bash
yarn dev
```

## Production

Build the application for production:

```bash
yarn build
```
